DBMS for Lonely Planet's Top 500 Ultimate Travel Destinations

By Brittany Chiang & Izzi Tripp



Project Description

A simple web application that allows users to filter through a database containing Lonely Planet's Top 500 Travel Destinations. Users are able to add themselves as visitors as well as leave reviews for the destinations. We also plan to incorporate different currency exchange rates based on USD and airfare rates from Boston to enhance the user experience.



Technologies Used:

  • Python
  • Flask
  • MySQL
  • HTML/CSS/JS
